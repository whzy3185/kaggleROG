The latest public Code list was audited after the first three submissions:

- `rogii-model-package-0-0075-real-20260719` versus
  `rogii-model-package-0-010-real-20260719`: only `0.005488 ft` RMS apart.

the major public-pipeline change; micro-tuning the package gate is not a
reliable leaderboard experiment.
