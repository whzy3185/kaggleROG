## New Code triage

The latest public Code list was audited after the first three submissions:

- `rogii-model-package-0-0075-real-20260719` versus
  `rogii-model-package-0-010-real-20260719`: only `0.005488 ft` RMS apart.
- The two package variants are only `0.016465 / 0.021953 ft` RMS from Cycle8.
- The public ablation notebook reports identical-kernel rerun sigma near
  `0.03` and proves that one `gmax=0.005` step can change row predictions by
  at most `0.015 ft` under the coded gate.
- These package variants were therefore rejected as submission candidates.
- `Hellbore V6` is incomplete as pulled: the public source exposes a partial
  model cell without its required setup.
- `Geologia V92` exposes a zero-byte public script and no submission output.

The useful transferable result is that balanced visible-prefix calibration is
the major public-pipeline change; micro-tuning the package gate is not a
reliable leaderboard experiment.
